# 悅和醬拼圖游戲

這是一個九塊拼圖圖游戲，玩家需要在60秒內將將打亂的拼圖圖碎片恢復成完整的圖片。

## 如何打開游戲

### 方法一：直接打開

1. 找到 `puzzle-game` 文件夾
2. 雙擊打開 `index.html` 文件
3. 游戲會在默認瀏覽器中運行

### 方法二：使用本地服務器

如果遇到圖片加載問題，請使用本地服務器：

1. 打開命令提示符或終端
2. 導航到 `puzzle-game` 文件夾：
   ```
   cd /path/to/puzzle-game
   ```
3. 運行本地服務器：
   ```
   python -m http.server 8000
   ```
4. 打開瀏覽器，訪問 http://localhost:8000

## 游戲功能

- 60秒倒計時
- 9塊拼圖碎片
- 成功完成後顯示獎勵界面
- 可隨時重置游戲

## 注意事項

- 請使用現代瀏覽器運行
- 確保網絡連接正常
- 如果圖片無法加載，請嘗試使用本地服務器方法
